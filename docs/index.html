AI Engineer Mastery

A production-shaped web app that turns a 68-lesson AI-engineering and system-design curriculum into a real product: accounts, a database, a spaced-repetition engine, weak-area analytics, and an in-app AI tutor that runs on any LLM provider you plug in. Built on the exact stack it teaches, so running and extending it is the lab.

FastAPI + async SQLAlchemy Postgres / pgvector FSRS-6 Groq / Claude / OpenAI-compatible Docker

Provenance

This app was extracted from Visakh Padmanabhan's resume-tailoring agent (Resume-app). The 68 lessons and the ~110-question interview bank (with answer keys) were authored there as a static study site and are bundled here in content/, ingested into the database on startup. The repo stands alone.

What you get

68-lesson curriculum

11 modules (LLM foundations → RAG → agents → evaluation → system design → interview mastery), rendered in-app with progress and streaks.

FSRS-6 spaced repetition

The ~110-question bank, with answer keys, scheduled by the 2026 state-of-the-art algorithm. Daily review queue, 4-button grading, per-card memory state.

AI tutor

Streamed Socratic teaching, scored mock interviews, and structured answer grading (score + the one fix + a model answer). Uses your notes as context.

Active recall

On any card, type your answer first and let the AI grade it before you reveal the key.

Analytics

Weak-area detection from graded attempts, strongest topics, a 30-day activity heatmap, streaks.

Runs anywhere

SQLite for zero-config local; Postgres + pgvector for production. Self-contained, $PORT-aware, deploys free.

Bring your own key — any provider

Pick one in backend/.env

The tutor is provider-agnostic and degrades gracefully with no key (the rest of the app still works).

# Groq — FREE, no credit card (https://console.groq.com)
LLM_PROVIDER=groq
GROQ_API_KEY=gsk_...
# …or Claude
LLM_PROVIDER=anthropic
ANTHROPIC_API_KEY=sk-ant-...
# …or ANY OpenAI-compatible host (OpenAI, Together, OpenRouter, local Ollama)
LLM_PROVIDER=groq
GROQ_API_KEY=<that host's key>
OPENAI_BASE_URL=https://your-host/v1
